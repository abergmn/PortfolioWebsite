# PortfolioWebsite

A personal portfolio website showcasing projects, contact info, and a basic admin interface.

---

## Features

- **About Me Page** — Personal introduction and details.
- **Projects Gallery** — Scrollable gallery with featured projects.
- **Contact Form** — Visitors can submit messages.
- **Admin Login Panel** — Access submitted messages via a login form.
- **Message Viewer** — Admin panel to view user-submitted messages.
- **Unique Visitor Counter** — Tracks unique visitors by IP.

---

## Setup Instructions

1. **Clone the repository**
    ```bash
    git clone https://github.com/abergmn/PortfolioWebsite.git
    cd PortfolioWebsite
    ```

2. **Run the built-in PHP server**
    ```bash
    php -S 0.0.0.0:8000
    ```

3. **Open website in web browser**
    ```
    Visit http://localhost:8000/
    ```
